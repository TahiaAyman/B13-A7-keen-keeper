
#  Project Name
  KeenKeeper — Keep Your Friendships Alive

# FOR RUN

cd keen-keeper
npm install
npm run dev

# Description

KeenKeeper is a modern web application that helps users manage and maintain meaningful relationships by tracking interactions such as calls, texts, and video chats. It ensures users stay connected and never miss important follow-ups.

# Technologies Used
Next.js (App Router)
React (TypeScript)
Tailwind CSS
Context API (State Management)
JSON (Mock Data)

# Features
Track interactions (Call, Text, Video)
Automatic timeline updates
Real-time popup notification on interaction
Friendship status monitoring (Overdue, On-track, Almost due)
Analytics dashboard with interaction insights
Clean and responsive user interface

###  Quick Check-In
- One-click interaction buttons:
  -  Call
  -  Text
  -  Video
- Automatically adds interaction to Timeline
- Instant popup notification 

###  Timeline Tracking
- View all interactions in one place
- Filter by:
  - Call
  - Text
  - Video
- Sorted with latest activity on top

##  Analytics (Stats Page)
- Visual chart showing:
  - Call count
  - Text count
  - Video count
- Dynamic updates based on Timeline data

###  Clean UI Design
- Minimal and modern layout
- Fully responsive design
- Smooth user experience
